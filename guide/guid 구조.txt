form = block
__group, __field-group = element

form
 └── form-group  ← (행 단위)
      ├── label
      └── form-field-group ← (필드 내부)
           ├── input
           ├── button
           └── icon



BEM 표기법
.block {}
.block__element {}
.block--modifier {} 


Block : 독립적으로 존재하는 구성요소
컴포넌트 단위
header, container, menu, checkbox, input
.btn / .input /.form / .table / .tabs / .tooltip / .card

Element : 블록의 부분이며 의미적으로 블록에 종속됨
내부 구성요소

menu item, list item, checkbox caption, header title
.btn__icon / .btn__text
.form__label / .form__input / .form__help
.table__head / .table__row / .table__cell

Modifier : 블록, 엘리멘트의 플래그이며 외형이나 행동을 바꿈
상태 / 변형

disabled, highlighted, checked, fixed, size big, color yellow
.btn--primary / .btn--secondary / .btn--lg / .btn--sm /
.tabs__button--active
.tooltip--open
