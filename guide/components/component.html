Button

개요

Button 컴포넌트는 사용자 동작을 실행하거나 다음 단계로 이동시키는 데 사용합니다. 링크와 버튼의 역할을 구분하고, 상태별 스타일과 접근성 기준을 함께 관리합니다.

기본 예시

<button type="button" class="btn">기본 버튼</button>
<button type="button" class="btn btn--primary">주요 버튼</button>
<button type="button" class="btn btn--secondary">보조 버튼</button>

사용 규칙

  • 페이지 이동은 a, 동작 실행은 button을 사용합니다.
  • Modifier는 색상, 크기, 상태처럼 시각적 또는 기능적 변형에만 사용합니다.
  • 텍스트 없는 아이콘 버튼은 대체 가능한 이름을 반드시 제공합니다.
  • 동일한 역할의 버튼은 프로젝트 전반에서 동일한 명칭과 상태 규칙을 유지합니다.

접근성 유의사항

이름 제공
버튼의 목적이 텍스트 또는 대체 텍스트로 명확히 전달되어야 합니다.
키보드 사용
Tab 이동과 Enter 또는 Space 조작이 가능해야 합니다.
상태 전달
비활성 상태는 시각적으로만 구분하지 말고 실제 disabled 상태도 함께 적용합니다.
